Forms

  • GET Form
  • POST Form
  • PUT Form
  • DELETE Form

Explanation: GET gets a message from the server given an id.

Explanation: POST creates a new message on the server. The id of the new message will be displayed afterwards.

Explanation: PUT updates a message on the server given the id and the new message.

Explanation: DELETE deletes a message on the server given the id.
